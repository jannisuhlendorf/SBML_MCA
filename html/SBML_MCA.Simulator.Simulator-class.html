find a steady state of the system, if the steady state before a parameter perturbation (old_ss), the parameter change (d_params) and the response coefficients of the unperturbed model (old_conc_resp) are given, the initial values are linearly approximated (can be much faster, than with initial values)

Parameters:
  • s0 (numpy.array) - vector of initial concentrations
  • tol (number) - tolerance
  • old_ss (numpy.array) - previously computed steady stae
  • d_params (numpy.array) - vector of parameter deviations
  • old_conc_resp (numpy.array) - the concentration response coefficients of the model with the old parameter values
  • string_output (boolean) - boolean indicating whether the output should be in string from
Returns: numpy.array
vector of steady state metabolite concentrations
